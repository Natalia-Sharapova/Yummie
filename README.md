# Yummie

## Добрый день, дорогие гости! 
## Хочу представить вам мини-приложение по заказу еды: Yummie, где я тренировала навыки создания xib, page control, сложных view и работы с URLSession.

## Были использованы: 
### -Xcode 12.5, 
### -UIKit, 
### -Podfile Kingfisher, позволяющий кешировать изображения, 
### -Podfile ProgressHUB, позволяющий визуализировать прогресс загрузки с сервера
### -Для настройки ячеек были использованы xib
### -@IBInspectable для доступа к свойству view через storyboard
